GARUDA GALLERY

Web development project using Python Django framework

Garuda Gallery is a responsive image hosting web app made as a school project with a team of 3 people during web development module. Garuda Gallery allows user to upload, group and view images. Search feature is still under development, so the user can only view their own images.

My main resposibilies in the project was to develop the image uploading and grouping. When uploading the app creates 2 new images with lower resolutions that are saved to the database. This means that the database holds 3 images for single upload (Org, MidRes, Thumbnail).
