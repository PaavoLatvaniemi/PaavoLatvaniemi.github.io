ROBOSHOCK

Game development project using Unity game engine and C# language

Roboshock is a arena shooter style first person shooter, that was made in 2,5 months as a school project by 2 people. The game can be played in matchmaking or in lan hosted sessions.
Our focus was to create fun game mechanics, that are easy to learn but hard to master. Roboshock's interesting movement is based on wall jumps. Player is able to lauch him/her self from any wall and keep jumping between walls. This combined with the warp mechanic that allows you to leep forward, gives player a total freedom to move strategically around the level.

My main resposibilies in the project was to do the level design for the game and network syncing between clients and the server. For networking framework I chose Unity's own MLAPI library. This was my first time doing multiplayer syncing and during this project I learned a lot about how data is transfered between server and all the clients.

I have a public repository of the game with only code in it. The main repository is privated due to usage of paid assets.
If you are interested check out our Roboshock movement showcase video, it shows a few demo scenarios of our movement.
