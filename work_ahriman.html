PROJECT AHRIMAN

Game development project using Unity game engine and C# language

Project Ahriman is a first person survival horror game, that was made in 7 weeks as a school project by 3 people. Our focus was on creating an AI enemy that feels alive and inteligent.
Player is guided through the game and introduced to the enemy alien aka "Skull Crusher" by story cutscenes. Player quickly learns that the only way to stay alive is sneaking and hiding from the alien.

Project Ahriman was inspired by Alien: Isolation game developed by The Creative Assebly in 2014. We took the base idea of an insta killing enemy alien and designed our own AI behaviour and wrote a story around it.
My main resposibilies in the project was to develop the AI behaviour tree and to do most of the level designing. For the behaviour tree I used code based state machine pattern, that worked pretty well for what we were aiming for.

I have a public repository of the game with only code in it. The main repository is privated due to usage of paid assets.
